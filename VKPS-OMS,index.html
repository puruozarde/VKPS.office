Connect to Google Cloud (Firebase) for real-time data sync across all devices

📋 One-time setup (5 minutes):
  1. Go to console.firebase.google.com
  2. Create a new project → Enable Firestore Database
  3. Go to Project Settings → Web App → Copy config keys below
  4. Set Firestore rules to allow read/write (for your firm's use)

Your Firebase config is stored locally and only used to connect to your firm's database.
